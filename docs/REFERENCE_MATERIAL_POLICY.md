# Reference Material Policy

All external/reference-only assets must live under:

- `reference/`

This includes:

- extracted SDK trees (for example `rvlsdk`)
- upstream source mirrors/snapshots (for example `ACGC-PC-Port`)
- manuals, reverse-engineering notes, and third-party comparison artifacts

`reference/` is intentionally git-ignored at the repository root so these files are never committed by accident.
